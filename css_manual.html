CSS 3

CSS 3 (Cascading Style Sheets Level 3) It is used with HTML to create content structure, with CSS3 being used to format structured content. It is responsible for font properties, colors, text alignments, graphics, background images, tables and other components.

This new technology delivers some interesting features that improve overall page appearance. It is not really a requirement for a page to function; however, online users can observe that such a page will look good, be simple, be navigable, and function flawlessly. This makes it easier for developers to design a page to leave a lasting impression on visitors and potential buyers who love online browsing.

BASIC CSS STYLING

  • background-color:

    Defines a color on background.

  • color:

    Defines a text color.

  • font-size:

    Defines the text size.

  • font-family:

    Defines the font to be used.

  • border:

    Defines a border around an HTML element.

  • padding:

    Defines a padding (space) between the text and the border.

  • margin:

    Defines a margin (space) outside the border.
